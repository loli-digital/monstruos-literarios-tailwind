# Monstruos Literarios

Página web sobre monstruos literarios como Drácula, Frankenstein, Cthulthu, Jekyll y Mr Hyde y el fantasma de la ópera.

La particularidad del proyecto es que al hacer scroll, hay un efecto parallax en las imágenes de los monstruos.
